Claire's Game Dev Site

I'm a software engineer working in games. This page is my personal noodlings in HTML5. Everything on this page is written from scratch in vanilla JavaScript, using no existing engines or libraries. Why would I do that? Because it's fun. And also because I find it easier to learn how to use such things when I've tried to tackle the problem on my own first.

Games

  • Raptor Quest (WIP demo)

    (Warning: features sound) A free-roaming platformer written from scratch in vanilla ES6. Includes a flexible component-based architecture, support for multiple scenes with async loading, tile-map levels, momenutum-based physics, and an edit mode.

  • Raptor Run

    (Warning: features sound) A basic little "infinite flapper" HTML5 game about a scientifically-accurate feathered velociraptor. Written in vanilla ES6.

  • Top-Down Adventure Game (WIP Demo)

    Early attempt at a top-down action/adventure game engine. Features normal-mapped dynamic lighting on the main character but it's CPU-based and therefore kinda slow on some devices.

  • Game Level Prototype

    (Warning: features sound) Babby's first HTML5 game. I think it has a certain naive charm. Desktop only.

    Buttery-smooth GPU-based lighting in WebGL using hand-painted normal maps. Big upgrade over the CPU-based lighting in my current top-down game demo. The light follows your mouse or touch.
