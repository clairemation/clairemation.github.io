Perturbing object paths using normal maps

For my game project I want the environments to have a subjective, painterly look, but I don't want them to feel like flat backdrops. So I thought of painting the environments in 2D, and using normal maps to create 3D terrain. I've never heard of normal maps being used for this purpose, but I think the effect could be really cool.

(All the assets were drawn in Photoshop. Nothing is 3d-modeled.)

Experiment 1: A sphere

A ball follows the contours of a sphere. This was my first attempt, in which I just calculated a faux penetration vector and subtracted it from the movement vector. I'm pretty sure the math is off, but it's visually convincing.

Experiment 2: A bunch of ramps

Here I had another idea: rotate the character's plane of movement to the plane defined by the surface normal. This seems to allow more precision.
