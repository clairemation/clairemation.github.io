Time's Prisoner

I made this little game demo a while ago. Now I'm working on making it as a "real" graphical game using the HTML5 canvas.

It's about a time traveler who's had a little accident. Now you have to help her brave prehistoric dangers to collect the scattered chrono crystals and return home.

Original demo level
  • tense, play-tested level design
  • awesome CC-licensed soundtrack courtesy of Kevin MacLeod

    • Note: Requires a desktop browser

New WIP sprite engine
  • smooth, non-tile-based movement in 8 directions
  • subtle natural physics including inertia, collision, and wall "sliding"
  • atmospheric hand-drawn graphics (by me)

    • Note: Currently requires desktop version of Safari, Firefox, Chrome, or Opera browsers
